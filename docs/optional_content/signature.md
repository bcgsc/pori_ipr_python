# Mutation Signatures

These are the scores of individual mutation signatures. These can include cosmic and/or custom
signatures. The nnls field is the non-negative least squares contribution of the given signature

```json
{
    "mutationSignature": [
        {
            "associations": "string",
            "nnls": 0.0571,
            "signature": "SBS1"
        }
    ]
}
```
