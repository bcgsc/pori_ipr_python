# Mutation Burden

Measure of the relative counts of various types of mutations compared to those from a given cohort
of samples. The "role" determines with comparator these values are calculated in reference to

```json
{
    "mutationBurden": [
        {
            "snv": 4,
            "snvTruncating": 0,
            "indels": 0,
            "indelsFrameshift": 0,
            "sv": 92,
            "svExpressed": 40,
            "snvPercentile": 4,
            "indelPercentile": 10,
            "svPercentile": 60,
            "role": "primary"
        }
    ],
}
```
